# Referencia tecnica fornecida por Labanino

Estes arquivos preservam, sem adaptacoes funcionais, os trechos enviados pelo
autor do fluxo que produzia os croquis corretos.

Eles documentam o contrato esperado:

1. extrair entidades do PROJETO PDF;
2. classificar condutores pela cor CAD;
3. preservar as coordenadas originais;
4. retirar mapa, cotas, tabelas e demais elementos irrelevantes;
5. marcar postes, transformadores, estruturas e equipamentos;
6. usar o projeto limpo como fonte geometrica da geracao.

Os imports ausentes foram reconstruidos na arvore `sistema/`. Os arquivos desta
pasta sao referencia imutavel e nao participam diretamente da aplicacao.
